This plugin allows you to display Instagram photos on a Wordpress blog.

== Description ==

Photo Blogstar allows you to simply display your Instagram photos on your Wordpress blog.

For further information and instructions please see the plugin page: http://thedalby.uk/photo-blogstar-wordpress-plugin/

== Installation ==

Install the plugin the same as any other WordPress plugin.  

- Then go to the plugin settings, which are found inside the Appearance menu on the admin interface.

- Enter the Instagram Client ID and User ID.  

- To get the Client ID, you need to visit instagram.com/developer and register your application.


== Frequently Asked Questions ==

= How do I find my Instagram User ID? =

The User ID is not the same as your username.  To find your User ID, go to jelled.com/instagram/lookup-user-id and lookup your username.
A User ID lookup function will be added to the Photo Blogstar plugin in a future release.
